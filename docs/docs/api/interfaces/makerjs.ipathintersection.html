Optional path1Angles

path1Angles: number[]
  • Defined in packages/maker.js/src/core/maker.ts:514

This Array property will only be defined if the first parameter passed to pathIntersection is either an Arc or a Circle. It contains the angles of intersection relative to the first path parameter. The length of the array may be either 1 or 2.
